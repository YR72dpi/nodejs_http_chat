# Http Nodejs Chat

## Installation

```bash
# Change directory on the HNC folder

# Install node modules
npm update
```

If yo want to test, open terminal.

In the fisrt one :
```bash
# For listen
npm run rx
```

In the second one :
```bash
# For listen
npm run tx
```

Write something on the first one and your message should appear on the second one.
